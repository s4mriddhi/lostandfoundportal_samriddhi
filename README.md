# 🔍 Smart Lost & Found Portal

A rapid-prototyped, full-scale responsive web application designed for hackathons. This portal translates structural low-fidelity wireframes into a clean, functional frontend interface using a bold, high-contrast **Retro-Brutalist design language** built purely with semantic HTML5, CSS3.

---

## 🎨 Design System & Aesthetic Constraints

This application strictly adheres to a high-contrast wireframe style:
* **Monochrome Palette:** Pure black (`#000000`) and white (`#FFFFFF`) with functional gray text indicators.
* **Brutalist Borders:** Sharp 90-degree container caps (`border-radius: 0px`) built with 2px/3px solid black outlines.
* **Hard Shadows:** Zero soft gradients. Elements utilize crisp, solid offset box shadows (`box-shadow: 4px 4px 0px 0px #000000`).
* **Typography:** Strict monospace execution utilizing system mono configurations (`font-family: monospace`) across forms and tabular data grids.
* **Action Designations:** High-priority execution buttons wrapped inside explicit text frames (e.g., `[ LOGIN ]`, `[ CLAIM THIS ITEM ]`).

---

## 🖥️ Screen Architecture

The application seamlessly manages page routing view changes across **5 core desktop screens**:

1. **Login Gate:** Centered structural access panel built with robust form elements.
2. **Central Dashboard:** Full-width command center equipped with global black navigation anchors, category filtering capsule matrices, and dynamic record tables.
3. **Report Lost Item Form:** Multi-input structural data grid featuring a customized drag-and-drop file upload target interface.
4. **Report Found Item Form:** Symmetrical discovery pipeline mapped with modified helper placeholder prompts.
5. **Item Details Screen:** Split-pane interface presenting structural technical logs paired with twin actionable communication buttons.

---

## 🚀 Tech Stack & Setup

* **Markup:** Semantic HTML5 Structure
* **Styling Engine:** Tailwind CSS (via official Play CDN or compiled CSS framework utility distribution)
* **Scripting Engine:** Vanilla JavaScript (ES6+ DOM manipulation and event handlers for screen toggles)
